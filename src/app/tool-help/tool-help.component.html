To use PDFSplit first select the pdf files you want to work with in the file selector by clicking on the 'Browse...' button. Every selected pdf will be associated with a number visible above the file selector, this number will be referred to as pdf id. For the sake of the examples on this page imagine the following pdfs are selected:

Selecting pages from a pdf

To select pages from a pdf file put the pdf id you want to select from in brackets followed by a ':' and the pages you wish to select.

Selects pages 1,2 and 3 from Tutorial.pdf.

Ranges of pages can be specified by giving a start and end pages connected with '~':

Selects pages 1,2,3 and 6,7 from Tutorial.pdf. Note that ranges are inclusive and that pdfs start from page 1 (and not 0) to be consistent with natural page numbering.

To exlude only certain pages from a pdf a '-' can be added in front of a page block:

Selects all pages except 1,2 and 3 from Tutorial.pdf.

Combining pdfs

To combine pdfs you can write their ids one after another:

Combines the two pdfs Taxes1.pdf and Taxes2.pdf in the order their pdf id is given.

Of course you can also select pages while doing so:

Combines Taxes1.pdf and the first 9 pages of Taxes2.pdf.
